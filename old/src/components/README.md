# Component Library

CVKG provides a suite of premium components designed to look and feel state-of-the-art.

## Primitive Views
- **Text**: Support for high-fidelity typography via `runic-text`.
- **Shape**: Rectangles, circles, and rounded-rects with direct GPU fill support.
- **Image**: High-performance texture sampling (Muspelheim compatible).

## Interactive Controls
- **Button**: Integrated with Sleipnir for physical feedback.
- **Slider/Toggle**: Precision inputs with customizable "Berserker" themes.
- **TextField**: High-fidelity text entry with integrated ShieldWall accessibility.

## Container Views
- **VStack/HStack/ZStack**: The building blocks of your layout.
- **NavigationStack**: Seamless shared-element transitions via Bifrost.
