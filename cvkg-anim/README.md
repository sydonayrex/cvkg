# cvkg-anim (Sleipnir)

**Sleipnir** is the physics-driven animation engine for CVKG.

## Features
- **RK4 Spring Solver**: Fourth-order Runge-Kutta integration for organic motion.
- **State Integration**: Seamlessly animate any state value in the CVKG graph.
- **High Performance**: Designed to run at 120Hz+ for buttery-smooth transitions.
