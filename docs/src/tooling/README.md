# Tooling & CLI

CVKG comes with a suite of tools to accelerate your development workflow.

## cvkg-cli
The main developer tool for managing CVKG projects.
- `cvkg init`: Scaffold a new project.
- `cvkg dev`: Start the high-fidelity dev server with HMR.
- `cvkg build`: Produce production-optimized bundles.

## cvkg-webkit-server
A specialized server for remote UI inspector support.
- Provides a WebSocket bridge for live scene graph inspection.
- Powers the CVKG debugger interface.
