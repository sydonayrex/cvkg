# Core Ecosystem

The heart of CVKG consists of the foundational traits and data structures that define the framework.

- **cvkg-core**: Defines the `View` trait, design tokens, and global environment.
- **cvkg-scene**: Manages the retained scene graph and diffing logic.
- **cvkg-layout**: Implements the flexbox-inspired layout engine.
- **cvkg-vdom**: Provides the virtual DOM representation for web and remote rendering.
